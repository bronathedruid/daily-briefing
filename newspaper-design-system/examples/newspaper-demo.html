Critical Zero-Day Discovered in Apache Struts Framework

Security Week 1 hour ago 9.5

Security researchers have identified a critical remote code execution vulnerability in Apache Struts 2, affecting millions of enterprise applications worldwide. The CVE-2026-4491 vulnerability allows unauthenticated attackers to execute arbitrary code on vulnerable systems through malformed HTTP requests.

Security Advisory
Organizations using Apache Struts are urged to immediately apply the emergency patch released at 14:30 UTC today. No workarounds are available for this vulnerability.
